# Mango Stream Deck

A customizable stream deck application for Windows.

## How to Use

1. Extract all files to a folder
2. Run `Mango Stream Deck.exe`
3. Right-click any button to customize it
4. Use Settings to configure grid size and theme

## Features

- Customizable button grid (3x3 to 5x5)
- Custom icons and actions
- Dark/Light theme support
- Keyboard shortcuts and application launching

## Requirements

- Windows 10 or later
- No Python installation required
