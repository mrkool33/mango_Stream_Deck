## QUICK START

1. Extract all files to a folder
2. Run "Mango Stream Deck.exe"
3. Right-click any button to customize it
4. Click X to minimize to system tray
5. Right-click tray icon to restore or quit

## FEATURES

✅ Customizable button grid (1-8 rows/columns)
✅ 5 Action types:
   • Open applications
   • Launch websites
   • Execute keyboard shortcuts
   • Type text automatically
   • Multi-action (coming soon)

✅ Visual customization:
   • Custom button colors

   • Text size and color
   • Opacity controls

✅ Dark/Light themes
✅ Configuration persistence
✅ System tray support (NEW!)

## SYSTEM TRAY

The app now minimizes to the system tray:
- Click X button → Minimizes to tray
- Double-click tray icon → Restore window
- Right-click tray icon → Menu options
  - Show: Restore window
  - Hide: Minimize to tray
  - Quit: Close completely

## REQUIREMENTS

- Windows 10 or later
- No installation required
- Standalone executable
